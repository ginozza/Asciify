# Asciify

## Description
Converts a given image (JPEG, PNG) into ASCII art with optional color support.

## Usage
1. Build the program:
```bash
go build -o asciify
```

2. Run the program:
```bash
./asciify -image <path-to-your-image>
```

3. Example:
```bash
./asciify -image example.png
```

## Requirements
- Go (https://golang.org/)
